#{extends 'main.html' /} #{set title:'Colophon' /} This website is constructed using the following technologies.
Play Framework
The web server code runs on top of this lean, fast, easy-to-program web framework. It is like Rails for the JVM.
Scala
All the server-side programming is in this expressive, clean statically-compiled language. It is like statically-compiled Ruby running on the JVM.
MongoDB
All the server-side persistent data is in this scalable document-oriented data store that supports geospatial indexing.
JQuery
The JavaScript code uses this JavaScript library to make working with the DOM simpler and cleaner.
JSLint
All the JavaScript passes this checker to make sure that it only includes the good bits of that language.
Datalicious Supercookies
The anonymous user tracking uses this JavaScript library to store the user's random ID across different browsers.
IP-based Location
When the accurate geolocation fails it falls back to using this web service to give approximate location.
Modernizr
The accurate geolation uses this JavaScript library to give a consistent standard API to modern browser capabilities.
Casbah
Our MongoDB access from Scala uses this Scala API.
